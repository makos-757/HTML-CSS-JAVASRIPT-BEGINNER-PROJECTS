## 🚀 Getting Started

1. Clone this repository
2. Navigate to any project folder
3. Open the `index.html` or `login.html` file in your browser
4. Explore the code to learn web development concepts

## 💻 How to Use

Each project is self-contained with:
- `index.html` or `login.html` - HTML structure
- `style.css` - Styling
- `main.js` - JavaScript functionality

Simply open the HTML file in a web browser to run the project.
